```
 * 6种原型数据类型

```
null和undefined:
>alert(null == undefined);  //输出 "true"
尽管这两个值相等，但它们的含义不同。
undefined 是声明了变量但未对其初始化时赋予该变量的值，  
null 则用于表示尚未存在的对象。如果函数或方法要返回的是对象，那么找不到该对象时，返回的通常是 null。
