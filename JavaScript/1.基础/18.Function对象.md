```javascript
// 定义一个函数
function doAdd(num){
    return num + 20;
}

// 使用Function创建刚刚的函数
var doAdd2 = new Function('num','return num+20');
console.log(doAdd2(20)); // 输出40

// 因为Funcation也是对象,所以也有属性和方法以及对象的公共属性和方法
// 尽管可以使用 Function 构造函数创建函数，但最好不要使用它，因为用它定义函数比用传统方式要慢得多。不过，所有函数都应看作 Function 类的实例。
```
