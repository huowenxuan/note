## Hexo
把note/下所有的文件夹放入hexo根目录/source/_posts/下

```
npm install -g hexo
// 将md转为html
hexo g 
// 开启hexo服务，监听4000端口
hexo server
```

注意

1. 不能有html文件，可以有md、图片和其他文件类型
2. 除了代码之外，其他地方不能有连续的两个`{`或`}`，加转移字符也没用，只能在中间加空格
3. 每个md文件的开头必须为：

    ```
    title: 标题
    ```
