### 输出对称数
打印出 1-1000000 之间的所有对称数(例如 121、1331 等)

```js
function judgeNum(start, end) {
    for (var i = start; i < end; i++) {
        var reversed = Number(i.toString().split("").reverse().join(""));
        if (reversed == i && i > 10) {
            console.log(reversed);
        }
    }
}
judgeNum(1, 1000000);
```
