## 环境变量

```
open ~/.bash_profile
// 添加
export PATH="/Users/huowenxuan/dev/flutter/bin:$PATH"
// 生效
source ~/.bash_profile
```
## 执行无反应，需要设置镜像
每次打开终端都要切换一下镜像

```
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```
