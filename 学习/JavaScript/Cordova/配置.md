## iOS图标、启动图
使用Perpo生成icon，将所有的icon放在`res/icon/ios`下，把所有的图标都在config.xml中标明，启动图同理

```
<platform name="ios">
  <icon height="60" src="res/icon/ios/icon-60.png" width="60"/>
  ...
  <splash height="1136" src="res/ios/splash/Default-568h@2x~iphone.png" width="640" />
  ...
</platform>
```
