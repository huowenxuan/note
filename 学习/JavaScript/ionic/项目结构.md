## src/index.html
App主要入口，运行App、设置脚本和CSS。

其中

```
<!-- 本地开发时在浏览器中报404错误，它会在Cordova打包时自动加到项目里 -->
<script src="cordova.js"></script> 
<!-- 自动生成，ts转为js -->
<script src="build/main.js"></script>
```
