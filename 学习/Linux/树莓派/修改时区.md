# 修改时区

```
# 查看时区 Mon, 01 Jul 2019 15:33:04 +0800。+0800为东8区
date -R 
# 选择时区，找到亚洲、中国、北京，确定，输出 Asia/Shanghai
tzselect
# 修改timezone文件
vim /etc/timezone
Asia/Shanghai
# 修改localtime
ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
# 或者
cp -a /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
```s
