---
title: 防火墙
---

# CentOS 7 

```
// 查看防火墙状态，running或not running
firewall-cmd --state 
// 关闭
systemctl stop firewalld.service
systemctl disable firewalld.service
```

#  CentOS 6

```
// 查看防火墙的状态：
service iptable status
// 临时关闭防火墙
servcie iptables stop 
// 永久关闭防火墙          
chkconfig iptables off
```
